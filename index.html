End of Service Notice

Tradamus is not creating new Editions. Despite interest in this technology and the reuse of many of its modules in other projects, the funding for this project has expired and the supporting technology has become brittle enough that maintenance costs overwhelmed demand. We apologize for any disappointment this creates and invite contact via email.

Importantly, the developers of this application remain very active in OSS and in developing Digital Humanities software resources. We are highly visible on Github and the code for this project has been archived there as well. We invite anyone to contribute or fork to this project and will enthusiastically support any individuals wanting to modularize, modernize, or recontextualize any of hte many tools contained within it.

If you have an individual project related to the goals of Tradamus (digital edition creation, simplified collation, text annotation), please do not be disheartened by this notice. Send us a message via email or complete an Intake Questionnaire and we'll reach out to you soon.
